# StatusSplit: Private Browser-Based Video Splitter

**🚀 Live Website:** [wappstatusplitter.site](https://wappstatusplitter.site/)

StatusSplit is a high-performance web tool built to solve the frustration of social media video limits. Designed and developed to provide a serverless video editing experience.

### ✨ Key Features
- **100% Client-Side:** Videos never leave your device. Your privacy is guaranteed.
- **No Watermarks:** High-quality exports for WhatsApp, Instagram, and TikTok.
- **Blazing Fast:** Powered by FFmpeg WASM for near-native performance.
- **Social Media Ready:** Presets for 30s (WhatsApp), 60s (Reels), and more.

### 🛠️ Technology Stack
- **WebAssembly (FFmpeg.wasm):** Professional-grade video engine in the browser.
- **Vanilla JavaScript:** Light-weight and high-performance logic.
- **Modern CSS:** Fully responsive and mobile-optimized UI.
